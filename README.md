# Soaplin
The Soaplin kernel is a new Unix-like operating system kernel.

## Features
* x86_64 support (kinda)
* Memory management (VMM/PMM)
* Simple pre-emptive scheduler
* Ring 3 (user mode) support

## To come

* ELF loader
* Virtual File System
* EXT2 driver
